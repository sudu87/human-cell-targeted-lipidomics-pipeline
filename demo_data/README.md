# Demo Data

This directory contains a small deterministic demo workbook for checking that the R environment and core SMS1/2 workflows run correctly.

The workbook is:

```text
sms12_demo_lipidomics.xlsx
```

It is generated by:

```sh
Rscript scripts/create_demo_datasets.R
```

The workbook contains four sheets:

- `sms12_total_demo`: raw SMS1/2-style total lipid data
- `sms12_individual_demo`: raw SMS1/2-style individual lipid data
- `pairwise_total_demo`: pairwise contrast results for total lipids
- `pairwise_individual_demo`: pairwise contrast results for individual lipids

These data are synthetic and are intended only for installation checks, demo runs, and reviewer testing. They should not be interpreted biologically.
